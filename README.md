## How to run

1.  Install All Dependencies

    # yarn install

2.  Create .env file in root directory
3.  Add environment variables to .env file

    # For NextAuth

        GOOGLE_ID

        GOOGLE_SECRET

        GITHUB_ID

        GITHUB_SECRET

        FACEBOOK_CLIENT_ID

        FACEBOOK_CLIENT_SECRET

        INSTAGRAM_CLIENT_ID

        INSTAGRAM_CLIENT_SECRET

    # For Stripe Checkout

        STRIPE_PUBLIC_KEY

        STRIPE_SECRET_KEY

        STRIPE_SIGNING_SECRET

4.  Run Project

    # yarn run dev
